Funcion para generar un número random:<br>

function getRandomInt(min,max){<br>
	return Math.floor(Math.random()*(max-min+1)+min)<br>
}<br>



Esto crea un número random entre el 1 y el 3:<br>

Math.floor(Math.random() * 3) + 1;<br>
